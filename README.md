# Photography Website (Full Stack)

A production-ready photography website with:
- Booking system
- Album & gallery management
- Admin panel
- Node.js backend
- PostgreSQL database
- HTML + CSS frontend

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express  
Database: PostgreSQL  
Auth: JWT  
Images: Cloudinary  

## Setup Instructions

### 1. Clone / Download
Download ZIP from GitHub or clone repository.

### 2. Backend Setup
```bash
cd backend
npm install
cp ../.env.example .env
npm start
